How to Play

DraftSim takes a given collection of cards and allows you to simulate draft and sealed games with it. To play, first create or import a collection. Next, select the format you wish to simulate.

Using the Collection Builder

The Collection Builder helps you create the starting card pool. You can search for existing cards by name, filtering the results by set, rarity, color, and more!
